# Progressive Enhancement App

This is a Vue 3 based app that displays posts from the r/Data subreddit. 
This only requires a recent version of the Docker Engine, 
or equivalent engine that supports the docker cli.

## Running the app
1. Clone repo to your machine
2. Using Terminal, or similar, from inside the cloned repo folder run `docker compose up -d`
3. Navigate to `localhost:8080` to view the app
